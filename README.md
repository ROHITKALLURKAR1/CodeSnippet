# ☕ Java CodeSnippets

A daily collection of small Java programs — one snippet at a time, every day.
The goal: build consistency, revisit core Java concepts, and create a quick-reference vault for the future.

## 📌 Why this repo exists
- Practice Java fundamentals and advanced topics in small, digestible pieces
- Build a daily coding habit
- Create a personal Java cheat-sheet I can search later

## 📂 Structure

```
java-codesnippets/
├── basics/            # variables, loops, conditionals, I/O
├── oops/               # classes, inheritance, interfaces, polymorphism
├── collections/         # ArrayList, HashMap, Set, etc.
├── streams/             # Java 8+ Stream API
├── multithreading/      # Threads, Runnable, Executors
├── algorithms/          # searching, sorting, recursion, DP
├── exceptions/           # try-catch, custom exceptions
├── io/                   # file handling, scanner, buffered reader
└── misc/                 # tips, tricks, gotchas
```

Each snippet is named like:
```
NNN-short-description.java
```
Example: `014-binary-search.java`

## 🗒️ Snippet format

Every file starts with a short header comment:

```java
/*
 * Topic: <short topic name>
 * Date: <YYYY-MM-DD>
 * Difficulty: Beginner / Intermediate / Advanced
 * Description: <one-line description>
 */
```
